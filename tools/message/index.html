GitHub Messaging App

App Usage Guide

  1. Login: Click "Log in with GitHub" to authenticate
  2. Upload JSON: Upload a configuration JSON with access token and hash groups
  3. JSON Editor: Modify your configuration directly in the editor
  4. Add Hash Groups: Create new hash groups for message categorization
  5. Search Messages: Find messages by hash group
  6. Submit Messages: Create new GitHub issues as messages
